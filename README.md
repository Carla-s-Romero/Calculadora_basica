# Calculadora

Uma calculadora web simples e funcional desenvolvida com HTML, CSS e JavaScript puro.

## 🎯 Características

- ✅ Operações matemáticas básicas (adição, subtração, multiplicação e divisão)
- ✅ Suporte a números decimais
- ✅ Parênteses para operações complexas
- ✅ Botão para limpar o display (C)
- ✅ Botão para deletar o último caractere (<<)
- ✅ Suporte a entrada via teclado (Enter para calcular)
- ✅ Interface responsiva e intuitiva

## 🚀 Como Usar

1. **Abra o arquivo** `index.html` em seu navegador
2. **Digite números e operadores** clicando nos botões ou usando o teclado
3. **Pressione Enter ou clique em "="** para ver o resultado
4. **Use "C"** para limpar o display completamente
5. **Use "<<"** para deletar o último caractere digitado

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e layout
- **JavaScript (Vanilla)** - Lógica e interatividade

## 📋 Operadores Suportados

| Símbolo | Operação |
|---------|----------|
| + | Adição |
| - | Subtração |
| X | Multiplicação |
| / | Divisão |
| ( ) | Parênteses |
| . | Ponto decimal |

## ⌨️ Atalhos de Teclado

| Tecla | Ação |
|-------|------|
| 0-9 | Digitar números |
| +, -, *, / | Operadores |
| ( ) | Parênteses |
| Enter | Calcular resultado |
| Backspace | Deletar último caractere |

## 💡 Exemplo de Uso

```
Entrada: 10 + 5 * 2
Pressione: Enter ou =
Resultado: 20
```

## 📝 Notas

- A calculadora utiliza a função `eval()` do JavaScript para processar as contas
- Validação de expressões inválidas com mensagens de alerta
- Interface simples e direta, focada em facilitar o uso
